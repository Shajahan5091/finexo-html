Prerequisites

  • A Snowflake account with the ACCOUNTADMIN role enabled.

Setup Guide for Snowflake

To set up the Snowflake Environment, you first need to create Snow Migrate-specific Snowflake entities (Warehouse, Database, Schema, User, and Role) with the OWNERSHIP permission to write data into Snowflake, track costs pertaining to Snow Migrate, and control permissions at a granular level. Follow these steps:

  • Log into your Snowflake account

  • Edit the upcoming script to change the password to a more secure password and to change the names of other resources if you desire.


Note:

Make sure you follow the Snowflake identifier requirements while renaming the resources.

You can use the following script in a new Snowflake worksheet to create the entities. Run the script using the Worksheet page or Snowsight.

Snowflake Script 

  -- set variables (these need to be uppercase)
  set snow_migrate_role = 'SNOW_MIGRATE_ROLE';
  set snow_migrate_username = 'SNOW_MIGRATE_USER';
  set snow_migrate_warehouse = 'SNOW_MIGRATE_WAREHOUSE';
  set snow_migrate_database = 'SNOW_MIGRATE_DATABASE';
    
  -- set user password
  set snow_migrate_password = 'password';
    
  begin;
    
  -- create SNOW_MIGRATE_ROLE
  use role securityadmin;
  create role if not exists identifier($snow_migrate_role);
  grant role identifier($snow_migrate_role) to role SYSADMIN;
    
  -- create SNOW_MIGRATE_USER
  create user if not exists identifier($snow_migrate_username)
  password = $snow_migrate_password
  default_role = $snow_migrate_role
  default_warehouse = $snow_migrate_warehouse;
    
    
  grant role identifier($snow_migrate_role) to user 
  identifier($snow_migrate_username);
    
  -- change role to sysadmin for warehouse / database steps
  use role sysadmin;
    
  -- create SNOW_MIGRATE_WAREHOUSE
  create warehouse if not exists identifier($snow_migrate_warehouse)
  warehouse_size = xsmall
  warehouse_type = standard
  auto_suspend = 60
  auto_resume = true
  initially_suspended = true;
  
  -- create SNOW_MIGRATE_DATABASE
  create database if not exists identifier($snow_migrate_database);
  
  -- grant warehouse access
  grant USAGE
  on warehouse identifier($snow_migrate_warehouse)
  to role identifier($snow_migrate_role);
  
  -- grant database access
  grant OWNERSHIP
  on database identifier($snow_migrate_database)
  to role identifier($snow_migrate_role);
  
  GRANT CREATE SCHEMA ON DATABASE SNOW_MIGRATE_DATABASE TO ROLE SNOW_MIGRATE_ROLE;
  
  USE ROLE ACCOUNTADMIN;
  
  GRANT CREATE INTEGRATION ON ACCOUNT TO ROLE SNOW_MIGRATE_ROLE;
  
  commit;
